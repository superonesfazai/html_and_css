#### CSS概述
```
CSS（ Cascading Style Sheets的缩写） ， 翻译为“层叠样式表”或者“级联样式表”， 简称样式表
```
#### 主要作⽤
```
1. 为了让⽹⻚元素的样式更加丰富， 也为了让⽹⻚的内容和样式能拆分开， CSS由此思想⽽诞⽣， CSS是
   Cascading Style Sheets 的⾸字⺟缩写， 意思是层叠样式表。 有了CSS， html中⼤部分表现样式的标签就废弃
   不⽤了， html只负责⽂档的结构和内容， 表现形式完全交给CSS， html⽂档变得更加简洁
2. 它主要是⽤来给HTML⽹⻚来设置外观或者样式
3. 外观或者样式： HTML⽹⻚中的⽂字的⼤⼩、 颜⾊、 字体， ⽹⻚的背景颜⾊、 背景图⽚
```
